# Ijebu Igbo Descendants in Diaspora

A premium, modern homepage for **Ijebu Igbo Descendants in Diaspora (IID)** — a global non-profit connecting Ijebu Igbo people worldwide.

![Homepage Screenshot](https://github.com/user-attachments/assets/2ae339d1-9819-47f9-95da-2cd0c24e2bdc)

## Overview

This website connects Ijebu Igbo people living around the world and promotes the development of Ijebu Igbo town in Ogun State, Nigeria.

## Tech Stack

- **Next.js 16** (App Router)
- **Tailwind CSS v4**
- **Framer Motion** (animations)
- **TypeScript**
- Responsive, mobile-first design

## Homepage Sections

1. **Navbar** — Fixed, animated navigation with mobile menu
2. **Hero** — Full-width hero with background image, headline, CTAs, and stats bar
3. **About** — Organisation mission with image, values grid, and quote
4. **Video Section** — YouTube embed celebrating Ojude Oba heritage
5. **Impact** — 4 impact cards: Community Development, Education, Cultural Preservation, Diaspora Networking
6. **Story** — Ijebu Igbo history, culture, festivals, and diaspora contribution
7. **Gallery** — Responsive masonry grid of cultural images
8. **CTA** — Newsletter signup and membership call-to-action
9. **Footer** — Social links, quick links, contact info, copyright
